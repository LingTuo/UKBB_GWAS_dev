Analysis of Asthma¶

This notebook applies the Get_Job_Script.ipynb to automatically generate the sbatch scripts to run in Yale's cluster. The end result is to apply various LMM workflows to perform association analysis in the ASTHMA trait, do clumping analysis and extract associated regions.
