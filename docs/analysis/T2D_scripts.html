Analysis of type 2 diabetes (T2D)¶

This notebook applies the Get_Job_Script.ipynb to automatically generate the sbatch scripts to run in Yale's cluster. The end result is to apply various LMM workflows to perform association analysis in the T2D trait, do clumping analysis and extract associated regions.

File paths on Yale cluster¶

  • Genotype files in PLINK format: /gpfs/gibbs/pi/dewan/data/UKBiobank/genotype_files/pleiotropy_geneticfiles/UKB_Caucasians_phenotypeindepqc120319_updated020720removedwithdrawnindiv
  • Genotype files in bgen format: SAY/dbgapstg/scratch/UKBiobank/genotype_files/ukb39554_imputeddataset/
